# วิธีส่งลิงค์ให้นักเรียนเข้าใช้เกม (แบบฟรี)

## ตัวเลือกที่ 1: ใช้ Vercel (แนะนำ – ฟรี)

1. **สมัคร Vercel** (ฟรี) ที่ [vercel.com](https://vercel.com) — ใช้ Google หรือ GitHub เข้าสู่ระบบได้

2. **อัปโหลดโปรเจกต์**
   - ไปที่ [vercel.com/new](https://vercel.com/new)
   - เลือก "Import Git Repository" ถ้าโปรเจกต์อยู่บน GitHub
   - หรือเลือก "Import" แล้วอัปโหลดโฟลเดอร์โปรเจกต์ (หรือลากโฟลเดอร์)

3. **ตั้งค่า Environment Variables**
   - ก่อน Deploy ให้เพิ่มตัวแปรตามไฟล์ `.env.example`:
     - `NEXT_PUBLIC_SUPABASE_URL` = ค่า URL จาก Supabase
     - `NEXT_PUBLIC_SUPABASE_ANON_KEY` = ค่า anon key จาก Supabase
     - `SUPABASE_SERVICE_ROLE_KEY` = ค่า service role key จาก Supabase
     - `ADMIN_PASSWORD` = รหัสผ่านครู (เช่น teacher2024)

4. **กด Deploy** — รอสักครู่ Vercel จะให้ลิงค์เช่น  
   `https://ชื่อโปรเจกต์.vercel.app`

5. **ลิงค์ที่ส่งให้นักเรียน**
   - **ลิงค์หน้าเข้าสู่ระบบนักเรียน:**  
     `https://ชื่อโปรเจกต์.vercel.app/login`
   - ส่งลิงค์นี้ใน Line / Google Classroom / ฯลฯ
   - นักเรียนเปิดลิงค์ → กรอกรหัสกลุ่มที่ครูให้ (เช่น group1, city1) → กดเข้าสู่ระบบ

6. **ลิงค์สำหรับครู**
   - ครูเข้า: `https://ชื่อโปรเจกต์.vercel.app/admin`  
   - ใช้ username และรหัสผ่านที่ตั้งในระบบ (เช่นใน `.env` หรือ Supabase)

---

## ตัวเลือกที่ 2: รันบนเครื่องตัวเอง + ใช้ ngrok (ฟรี)

ถ้าไม่อยากใช้ Vercel แต่รันบนคอมครูที่โรงเรียน:

1. รันเกมบนเครื่อง: `npm run dev`
2. ติดตั้ง [ngrok](https://ngrok.com) (มีแผนฟรี)
3. ใน terminal อีกหน้าต่างรัน: `ngrok http 3000`
4. ngrok จะให้ลิงค์เช่น `https://xxxx.ngrok.io`
5. **ลิงค์ส่งนักเรียน:** `https://xxxx.ngrok.io/login`  
   (หมายเหตุ: ลิงค์ ngrok ฟรีจะเปลี่ยนทุกครั้งที่รันใหม่)

## สรุปลิงค์ที่ใช้

| ใครใช้ | ลิงค์ |
|--------|--------|
| **นักเรียน** | `https://ชื่อโปรเจกต์.vercel.app/login` → กรอกรหัสกลุ่ม |
| **ครู** | `https://ชื่อโปรเจกต์.vercel.app/admin` → เข้าสู่ระบบครู |

ครูสร้างเมืองและกำหนดรหัสกลุ่ม (เช่น group1) ในแผงควบคุม แล้วแจกรหัสกลุ่มนี้ให้นักเรียนแต่ละกลุ่มไปกรอกที่หน้า login ข้างต้น
