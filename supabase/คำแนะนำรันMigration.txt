ถ้าเจอ 500 (Internal Server Error) หรือหน้ารายการเมืองไม่โหลด:

1. เปิด Supabase Dashboard → โปรเจกต์ของคุณ
2. ไปที่ SQL Editor
3. เปิดไฟล์ "รันไฟล์นี้ใน_Supabase_SQL_Editor.sql" ในโปรเจกต์นี้
4. Copy เนื้อหาทั้งหมดไปวางใน SQL Editor
5. กด Run
6. รอจนขึ้น Success
7. กลับมา Refresh หน้าแอดมิน / หน้าหลักนักเรียน

หลังรันแล้ว จะได้:
- คอลัมน์และตารางสำหรับฟีเจอร์เต็ม (เป้าหมายเจรจา, สินค้า 2 อย่าง, สงครามตามทรัพยากร, ค่าชดใช้ ฯลฯ)
- ทรัพยากร 12 แบบ: Rice, Weapons, Gold, Soldiers, ปืนใหญ่, อิฐสร้างเมือง, กองคาราวานไม้สักทอง, กองกำลังพรานป่า, ทองคำและเครื่องเทศ, เรือสำเภาค้าขาย, เรือรบติดปืนใหญ่, เสบียงอาหารทะเลแห้ง

ครูสามารถไปที่แท็บ เมือง → เลือกกลุ่ม → กำหนดจำนวนสินค้าแต่ละประเภทได้ แล้วกด "บันทึกทรัพยากร"
